# 属性

# Props

参数 描述 类型 可选值 默认值
limit 其余的按钮 number - 3
more-text 更多按钮文案 string - 更多
popover-class 为 popover 添加类名 string - -
trigger popover 触发方式 string click/focus/hover/manual click

# Slot

插槽 描述
more 更多按钮自定义
